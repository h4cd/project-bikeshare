Este proyecto tiene como finalidad aprender a crear, modificar y gestionar repositorios locales y remotos por medio de 
Git y GitHub.

Adicionalmente se aprendio a consultar las modificaciones que se hacian sobre los archivos y crear ramas para manejar
varias versiones en las cuales se pueden hacer modificaciones sin riesgos de dañar los archivos orginales.

El conocimiento adquirido se utliza para crear un repositorio en GitHub en donde se publica los archivos de proyectos
anteriores.
